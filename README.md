# Projeto Global Solution

Análise de Dados de Diversidade, Igualdade e Inclusão

Em resposta ao desafio do segundo semestre do curso de Data Science, desenvolvemos um produto direcionado para empregabilidade, principalmente de minorias sociais: o Plural.

Plural vai além de um código python, é uma visão ampla de diversidade da empresa, contando com análises de escolaridade, salário, atendimento de cotas, representatividade em cargos de 
liderança, etc.

Neste repositório realizamos algumas análises de uma empresa fictícia para indicar possíveis pontos de melhoria para atender a diversidade brasileira, essas análises seriam disponibilizadas 
em tempo real para o Departamento de Recursos Humanos via dashboard, contando com KPIs como Índice de Diversidade, Número de Pessoas para atingir a Cota, Proporção de Colaboradores por Escolaridade, 
entre outros.
