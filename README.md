# Projeto Global Solution

Análise de Dados de Diversidade, Igualdade e Inclusão

Em resposta ao desafio do segundo semestre do curso de Data Science, desenvolvemos um produto direcionado para empregabilidade, principalmente de minorias sociais: o Plural.

Plural vai além de um código python, é uma visão ampla de diversidade da empresa, contando com análises de escolaridade, salário, atendimento de cotas, representatividade em cargos de liderança, etc.

Neste repositório realizamos algumas análises de uma empresa fictícia, com o objetivo de indicar possíveis pontos de melhoria, e assim oferecer oportunidades com foco em inclusão. 

Essas análises seriam disponibilizadas em tempo real para o Departamento de Recursos Humanos via dashboard, contando com KPIs como:
- Índice de Diversidade: indicador de presença de grupos diversos.
- Número de Pessoas para atingir a Cota: mensurador de volume.
- Proporção de Colaboradores por Escolaridade: importante para direcionamento de treinamentos e benefícios relacionados à formação.
- Indicadores de diversidade com o tempo: fundamental para identificar se está melhorando ou piorando.
